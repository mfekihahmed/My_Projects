Ici vous pouvez créer un nouveau réseau.
En complément, un sous-réseau associé à ce réseau peut être créé dans le panneau suivant.
les adresses réseaux doivent respecter le format CIDR (i.e. 192.168.0.0/24)
Adresse IP de la passerelle (e.g. 192.168.0.254). La valeur par défaut est la première adresse IP du réseau (i.e. 192.168.0.1 pour 192.168.0.0/24). Si vous voulez utiliser l&#39;adresse par défaut, laissez tel quel. Si vous ne voulez pas utiliser de passerelle, cochez &quot;Désactiver la passerelle&quot; ci-dessous.

Vous pouvez créer un sous-réseau associé au nouveau réseau. Dans ce cas le paramètre "Adresse réseau" doit être spécifié. Si vous voulez créer un réseau sans sous-réseau, décochez la case "Créer un sous-réseau".
IP address allocation pools. Each entry is: start_ip_address,end_ip_address (e.g., 192.168.1.100,192.168.1.120) and one entry per line.
Liste des adresses IP des serveurs DNS pour ce sous-réseau. Une seule adresse par ligne.
Additional routes announced to the hosts. Each entry is: destination_cidr,nexthop (e.g., 192.168.200.0/24,10.56.1.254) and one entry per line.

Vous pouvez spécifier des paramètres supplémentaires pour le sous-réseau.
